
# tsrun

Run Typescript (`*.ts`) files from the shell without having to pre-compile to Javascript.  Useful for quickly writing command-line tools.

## install

```sh
$ npm install -g tsrun
```

## use

```sh
$ tsrun /foo/bar/xyzzy.ts
```
